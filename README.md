## Getting Started

To get a local copy up and running follow these simple example steps.

### Prerequisites

- Have a Desktop or Laptop Computer running on with Windows, Mac OS or Linux operating system.
- Have git installed on the Computer
- Have a browser install on the Computer

### Setup

- If it suits you, on your local computer, create a directory in your preferred location (e.g in the Desktop or Document directory)

### Install
- To clone the project on your machine: Type `git clone https://github.com/NahnahAJ/Trial-Project.git` if your're using https with git or `git@github.com:NahnahAJ/Trial-Project.git` if you prefer ssh and hit enter.
- To install project dependencies _(a packet manager is required)_:
`npm install`
- To build project files in development mode _(the destination folder is /dist)_:
`npm run build`
- To run the project on localhost:3000
`npm run start`
 

### Usage

- You can view the document in your preferred or default browser.
